Extracts the specified hint from the filename and returns it. If the hint was not present, an empty optional is returned instead.

Extracts the specified hint from the filename and returns it. If the hint was not present, an empty optional is returned instead. A hint is a character-sequence of the form [key-value] within the filename. For example, executing getHint("myhint") on a file named myfilename [myhint-the value] will return the value.

Parameters

Returns
Empty, if the key was not found. The value of the hint otherwise.

◆ getName()

std::string mogli::lib::GameEntry::getName ( ) const
noexcept

Returns the name of the game as it is inferred from the file name.

Returns the name of the game as it is inferred from the file name. The game's name is the initial substring that does not consist of the release year or any hints. For example, executing getName() on a file named myfilename [myhint-the value] (2009) will return myfilename.

Returns
The name of the game.
Note
getName() is the only exception where the info.yaml takes precedence over the filename.

Returns the year of the game's release as it is inferred from the file name. The year of the game's release is inferred by a substring satisfying the regex \(\d{4}\). For example, executing getReleaseYear() on a file named myfilename [myhint-the value] (2009) will return 2009.
